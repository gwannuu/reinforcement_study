\begin{frame}{Idea}
    \itemize{
        \item More generalize for unseen state. In policy learning, sampling not only $s$, but also $s^{\prime}$. Add maximize term $\max_V \mbb{E}_{s \sim \mc{D}}[V(s)] = \mbb{E}_{s \sim \mc{D}, a \sim \pi^k, a^\prime \sim \operatorname{Unif}}[Q(s,a) - \log \pi(a^\prime|s)]$ of CQL policy evaluation. Want to maximize value function for seen state $s \sim \mc{D}$.
        \item Modify Penalize term (current is $Q^\pi(s,a) \leftarrow \mc{B}^\pi Q^\pi(s,a) -\alpha \lvert \frac{\mu(a|s)}{\beta(a|s)} -1 \rvert$. What if this term changes action sampling N $a_1, a_2, \dots, a_N \sim \mu(\cdot|s)$ and $\sum_n \lvert a_n - a \rvert^2 /N$). This approach wants to basically wasserstein-2 distance of Flow Q Learning. Note that target domain is continuous control. And also, CQL implementation set $\mu$ not uniform distribution, instead mixed density of policy $\pi$ and $\mu$.
    }
\end{frame}
